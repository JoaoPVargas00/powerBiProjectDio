📊 Power BI – Sales & Profit Dashboard
Projeto desenvolvido como parte da Formação Power BI Analyst (Desafio de Projeto – DIO), com o objetivo de analisar vendas, lucros, segmentos, produtos e países, utilizando recursos avançados de visualização no Power BI.
🧠 Objetivo do Projeto
Criar dashboards interativos e intuitivos para:

Acompanhar indicadores-chave de vendas
Analisar lucro por produto, país, segmento e período
Facilitar a tomada de decisão baseada em dados
Aplicar boas práticas de modelagem e visualização no Power BI


📈 Dashboards Desenvolvidos
🔹 Sales Report
Dashboard focado na análise de vendas, contendo:

KPIs principais:

Total de Vendas
Unidades Vendidas
Soma de Descontos
COGS


Evolução mensal das vendas
Vendas por segmento
Vendas por produto
Análise geográfica (mapa por país)
Filtro interativo por data
